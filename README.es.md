# Nack Tracing: un motor de trazado de rayos fotorrealista en C#

Este proyecto, denominado **Nack Tracing**, es el Trabajo de Fin de Grado (TFG) de la Escuela de Ingeniería Informática de la Universidad de Oviedo. Consiste en el diseño y la implementación de un motor de trazado de rayos desde cero utilizando C# y la plataforma .NET.

El motor es capaz de generar imágenes sintéticas con un alto grado de realismo visual mediante la simulación del comportamiento físico de la luz al interactuar con diferentes materiales y geometrías en una escena 3D.

El desarrollo del proyecto se guía por dos objetivos principales:
**Arquitectura de software:** construir una arquitectura de software robusta y escalable mediante la aplicación de patrones de diseño orientados a objetos para gestionar eficazmente la complejidad de la escena 3D.
**Optimización del rendimiento:** abordar el alto coste computacional inherente al trazado de rayos mediante la implementación de optimizaciones de rendimiento tanto a nivel de algoritmo como de código, con el objetivo de mejorar la eficiencia y reducir los tiempos de renderizado.

## 🛠️ Tecnologías
*   **Lenguaje:** C#
*   **Plataforma:** .NET
*   **IDE:** Visual Studio 2026
